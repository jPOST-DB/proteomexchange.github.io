Mission

The ProteomeXchange Consortium has been set up to provide a globally coordinated submission of mass spectrometry proteomics data to the main existing proteomics repositories, and to encourage optimal data dissemination. Please review our Data Submission Guidelines and PX Membership Agreement. See also the summary Nature Biotechnology paper published in 2014.

Public Data

Public PXD datasets can be browsed over at ProteomeCentral. An RSS feed is also available.

Member repositories and data submission

The founding members of ProteomeXchange are PRIDE, the PRoteomics IDEntifications Database at the European Bioinformatics Institute, based in Hinxton, UK and PeptideAtlas, part of the Institute of Systems Biology in Seattle, USA. Since the launch MassIVE has joined as well and other members will join soon. ProteomeXchange fully supports both MS/MS proteomics and SRM data submission. Submissions of other types of proteomics data is also possible using the Partial Submission mechanism.

PRIDE - PRoteomics IDEntifications Database

PRIDE accepts both Complete Submissions, based on standard formats and Partial Submissions as well. The Submission Guidelines provide the necessary information from start to finish. The GUI based PX Submission Tool can be downloaded to start data upload. The FAQ can be consulted for particular questions. Detailed information can be seen in the Submission Summary File Format and in the Submission Tutorial documents.

PeptideAtlas - PASSEL

SRM/MRM data can be submitted to PASSEL.

MassIVE

Shotgun proteomics data can be submitted to MassIVE.
